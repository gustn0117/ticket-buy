02

관리자 패널 사용법

/admin 접속 후 비밀번호 1234를 입력하면 관리자 패널에 진입합니다.

방문자 통계

상단에 오늘 방문자 수, 누적 방문자 수, 최근 30일 바 그래프가 표시됩니다.

회원관리

  • 전체 회원 목록 확인 (이름, 이메일, 연락처, 가입일)
  • 유형 전환: "일반" 또는 "업체" 뱃지를 클릭하면 유형이 전환됩니다
  • 회원 삭제 (휴지통 아이콘)

게시글관리

  • 전체 게시글 목록 (판매/구매 구분, 카테고리, 가격, 작성자, 조회수)
  • 게시글 삭제

공지관리

  • 상단 입력란에 공지 제목 입력 후 "등록" 클릭
  • "고정" 체크 시 공지 목록 최상단에 고정 표시
  • 공지 삭제

거래관리 (채팅)

  • 전체 채팅/거래 목록 (게시글, 구매자, 판매자, 단계, 상태)
  • 채팅 클릭 시 우측에 실시간 메시지 뷰어가 열립니다
  • "중개" 뱃지가 표시된 거래는 중개 거래입니다
  • 채팅 삭제 (관련 메시지도 함께 삭제됨)
TIP 실시간 뷰어에서 새 메시지가 자동으로 갱신됩니다. 분쟁 발생 시 거래 내역을 즉시 확인할 수 있습니다.
03

프리미엄 업체 관리

관리자 패널 > 프리미엄 탭에서 프리미엄 구매 업체를 관리합니다.

업체 등록 방법

1"새 업체" 버튼 클릭
등록 폼이 나타납니다.
2업체 정보 입력
업체명(필수), 소개글, 연락처, 지역, 취급 브랜드(쉼표로 구분), 업체 이미지(업로드)
3광고 등급 선택
프리미엄 (최상위 노출) / 스탠다드 (일반) / 베이직 (기본)
4연결 회원 선택 (선택사항)
업체 회원을 연결하면 해당 회원의 구매글이 업체 상세 페이지에 표시됩니다.
5"등록" 클릭
메인 페이지와 게시판의 "삽니다" 탭에 즉시 노출됩니다.

노출 위치

  • 메인 페이지 > 삽니다 탭 > 상단 2열 카드 영역
  • 게시판 페이지 > 삽니다 탭 > 상단 2열 카드 영역
  • 등급별 탭 필터 (전체/프리미엄/스탠다드/베이직)
  • 카드 클릭 시 업체 전용 상세 페이지로 이동
04

광고 관리

관리자 패널 > 광고 탭에서 배너 광고를 관리합니다.

광고 등록 방법

1"새 광고" 버튼 클릭
2슬롯 위치 선택
메인 히어로 배너, 메인 상단, 게시판 상단, 게시글 하단, 푸터, 팝업 등
3광고 정보 입력
제목, 설명, 광고주명, 이미지(업로드), 클릭 시 이동 URL
4기간 및 우선순위 설정
시작일/종료일, 같은 슬롯에 여러 광고가 있으면 우선순위가 높은 것이 표시됩니다.
TIP 광고가 등록되지 않은 슬롯에는 빗금 패턴과 슬롯명/사이즈가 표시됩니다. 이를 통해 어떤 위치에 광고를 넣을 수 있는지 직관적으로 확인할 수 있습니다.
05

거래 흐름

직접 거래 (6단계)

구매자와 판매자가 직접 거래합니다. 플랫폼이 단계를 안내합니다.

1견적
구매자가 수량, 금액, 발송일을 제안합니다.
2기본정보
판매자가 기본정보를 제출하고, 구매자가 확인합니다.
3계약서
전자 계약서를 작성합니다.
4입금
구매자가 판매자에게 직접 입금합니다.
5배송
판매자가 상품권을 전달합니다.
6완료
구매자가 수령을 확인하면 거래가 완료됩니다.

중개 거래 (8단계)

티켓바이가 중간에서 대행합니다. 게시글 상세에서 "중개거래로 안전하게" 버튼으로 시작합니다.

1~3견적 → 기본정보 → 중개 계약서
4구매자 → 중개업체 송금
5중개업체 → 판매자 송금
6판매자 상품권 → 중개업체 수령
7중개업체 → 구매자 전달
8거래 완료

채팅

PC에서는 우측 하단 플로팅 위젯으로, 모바일에서는 하단 "채팅" 버튼으로 이용합니다. 게시글에서 구매하기를 누르면 채팅이 자동 생성되고 위젯이 열립니다.

글쓰기

  • 로그인 후 메인 또는 게시판의 "글쓰기" 버튼 클릭
  • 글 유형 (판매/구매), 상품권 종류, 제목, 금액 입력
  • 금액 입력 시 자동으로 쉼표(,)가 붙습니다
  • 자신의 글은 상세 페이지에서 수정/삭제 가능

회원 유형

일반 회원

회원가입 페이지(/register)에서 가입. 판매/구매 글 작성 및 채팅 이용 가능.

업체 회원

업체등록문의(/register-business) 신청 후 관리자가 승인. 사업자번호 기반 로그인.

비회원

회원가입 없이도 게시글 열람, 프리미엄 업체 연락처 확인이 가능합니다. 글 작성과 채팅은 로그인이 필요합니다.
