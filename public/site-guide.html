1. 사이트 소개

티켓바이는 상품권을 사고파는 사람들을 연결해주는 중개 플랫폼입니다.

판매자가 상품권을 올리면 구매를 원하는 사람이 연락해서 거래가 이루어지는 구조이고, 반대로 구매 업체가 "이런 상품권을 삽니다"라고 올릴 수도 있습니다.

거래 과정에서 계약서를 자동으로 만들어주고, 단계별로 진행 상황을 관리해줘서 사기 피해를 줄일 수 있도록 했습니다. 더 안전하게 하고 싶은 분은 중개 거래를 선택할 수 있는데, 이 경우 티켓바이가 중간에서 돈과 상품권을 직접 전달해줍니다.

주요 기능 요약

  • 삽니다 / 팝니다 게시글 등록 및 조회
  • 실시간 채팅 (PC에서는 우하단 위젯, 모바일에서는 하단 버튼)
  • 직접 거래 / 중개 거래 선택
  • 프리미엄 구매 업체 카드 노출 (등급별 차등)
  • 배너 광고 시스템 (11개 슬롯)
  • 관리자 패널 (회원, 게시글, 거래, 광고, 통계 관리)

취급 상품권

롯데, 신세계, 문화상품권, 컬쳐랜드, 스타벅스, 온캐시, 구글플레이, 틴캐시, 배민상품권, 해피머니 등

회원 종류

일반 회원 - 회원가입 페이지에서 가입합니다. 글 작성, 채팅 이용이 가능합니다.

업체 회원 - 업체등록 신청을 하면 관리자가 확인 후 승인합니다. 사업자번호로 로그인합니다.

비회원 - 가입 없이도 게시글 열람, 프리미엄 업체 연락처 확인이 가능합니다. 글 작성과 채팅만 로그인이 필요합니다.

3. 관리자 패널

/admin 에 접속해서 비밀번호 1234를 입력하면 들어갈 수 있습니다.

들어가면 상단에 오늘 방문자 수, 누적 방문자 수가 나오고, 30일간 방문자 추이를 막대 그래프로 볼 수 있습니다.

회원관리

가입한 회원 목록이 나옵니다. "일반"이나 "업체" 글씨를 누르면 유형이 바뀝니다. 오른쪽 휴지통 아이콘으로 삭제할 수 있습니다.

게시글관리

등록된 모든 게시글을 볼 수 있습니다. 판매/구매 구분, 카테고리, 가격, 조회수 등이 표시되고, 문제가 있는 글은 삭제할 수 있습니다.

공지관리

상단 입력창에 공지 제목을 쓰고 "등록"을 누르면 됩니다. "고정"에 체크하면 목록 맨 위에 고정됩니다.

거래관리

진행 중인 채팅/거래 목록이 나옵니다. 아무 거래나 클릭하면 오른쪽에 대화 내용이 실시간으로 보입니다. "중개"라고 표시된 건 중개 거래입니다.

참고 채팅을 삭제하면 해당 대화의 모든 메시지도 같이 삭제됩니다. 분쟁이 있는 경우에는 삭제 전에 내용을 먼저 확인하세요.

프리미엄 업체

"새 업체" 버튼을 눌러서 등록합니다. 업체명, 소개글, 연락처, 지역, 브랜드를 입력하고, 이미지는 직접 업로드할 수 있습니다.

등급은 세 가지입니다.

  • 프리미엄 - 가장 위에 노출됩니다
  • 스탠다드 - 일반적인 상위 노출입니다
  • 베이직 - 기본 노출입니다

"활성/비활성" 버튼으로 노출을 켜고 끌 수 있습니다. 회원 연결을 하면 그 업체 회원이 올린 구매글이 업체 상세 페이지에 같이 보입니다.

광고관리

"새 광고" 버튼을 눌러서 등록합니다. 어떤 위치에 넣을지 선택하고, 이미지를 업로드하고, 클릭하면 이동할 주소를 넣으면 됩니다. 기간도 설정할 수 있고, 같은 자리에 여러 개 넣으면 우선순위가 높은 게 보입니다.

4. 거래 진행 방법

직접 거래

게시글 상세에서 "구매하기 (직접거래)" 버튼을 누르면 채팅이 만들어집니다. 이후 아래 순서로 진행됩니다.

  1. 견적 - 구매자가 수량, 금액, 발송일을 제안합니다
  2. 기본정보 - 판매자가 정보를 제출하고 구매자가 확인합니다
  3. 계약서 - 계약서를 작성합니다
  4. 입금 - 구매자가 판매자에게 입금합니다
  5. 배송 - 판매자가 상품권을 보냅니다
  6. 완료 - 받은 걸 확인하면 거래가 끝납니다

중개 거래

"중개거래로 안전하게" 버튼을 누르면 티켓바이가 중간에서 대행합니다. 돈도 상품권도 티켓바이를 거쳐서 전달되기 때문에 직접 거래보다 안전합니다.

  1. 견적
  2. 기본정보
  3. 중개 계약서 작성
  4. 구매자가 중개업체 계좌로 송금
  5. 중개업체가 판매자에게 송금
  6. 판매자가 상품권을 중개업체에 전달
  7. 중개업체가 구매자에게 상품권 전달
  8. 거래 완료

채팅 사용법

PC에서는 화면 오른쪽 아래에 검은 동그라미 버튼이 있습니다. 누르면 채팅 목록이 나오고, 대화방을 선택해서 메시지를 주고받을 수 있습니다.

모바일에서는 하단 메뉴의 "채팅" 버튼을 누르면 같은 위젯이 열립니다.

게시글에서 "구매하기"를 누르면 채팅이 자동으로 만들어지면서 위젯이 열립니다.
